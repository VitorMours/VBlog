# VBlog 

O VBlog é um projeto desenvolvido em **Django** com o objetivo de ser um caso de estudo sobre:

- Autenticação e Autorização dentro do **Django**
- Processamento de dados usando o **pandas** e **numpy**
- Armazenamento de Dados persistentemente com **MySQL**
- Containarização e virtualização com **Docker**
- Interatividade com **Javascript**
- Comunicação com outros apps por meio de **API** e **requests**
